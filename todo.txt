Environment (robot_environment.py)

Interfaces with the NAO hardware
Translates actions to robot commands
Collects sensor data and provides observations


Agent (robot_agent.py)
Receives input commands (from keyboard/GUI)
Makes decisions about robot actions
Will later contain your reinforcement learning logic


GUI (robot_gui.py)

Simple interface for controlling the robot
Displays robot state information


Main Controller (main.py)

Connects all components together
Handles program lifecycle
